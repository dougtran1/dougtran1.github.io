Saigon's Corner

UX Designer
Penpot, Adobe XD
Website Redesign
[ Add project image here ]

Overview

Saigon's Corner is a website redesign project for a local Vietnamese restaurant. The goal was to improve the user experience, modernize the visual design, and make it easier for customers to find information and place orders.

The Problem

The existing site had an outdated design, poor mobile experience, and unclear navigation. Customers were struggling to find the menu, hours, and contact information — leading to a drop in engagement.

Process

  • Conducted informal user research and heuristic evaluation of the existing site
  • Identified key pain points: navigation structure, menu legibility, mobile layout
  • Created low-fidelity wireframes to map out the new information architecture
  • Developed a high-fidelity prototype in Penpot with updated visual design
  • Documented findings and design decisions in a written case study

Key Design Decisions

  • Simplified navigation to 4 core sections: Menu, About, Gallery, Contact
  • Introduced a warm, food-photography-forward hero section
  • Mobile-first layout with tap-friendly menu browsing
  • Consistent type hierarchy to improve scanning and readability

Outcome

The redesign prototype demonstrates a clear improvement in usability and visual quality. This project showcases my ability to take a design from research through wireframes to a polished prototype.
